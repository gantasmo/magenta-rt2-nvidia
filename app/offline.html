The Studio engine didn’t start

This almost always means setup hasn’t finished, or WSL/Ubuntu needs a nudge. The fix is the same friendly installer you ran before. It’s safe to run again and skips anything already done.

  1. Go back to the folder you unzipped and double-click Setup-MRT2.bat.
  2. It re-checks your PC, repairs what’s missing, and offers to launch the Studio for you.
  3. If it reports a blocker (e.g. no NVIDIA GPU), follow the on-screen step, then run it again.

Still stuck after that? Open INSTALL.md in the same folder. It has a short “If something goes wrong” list.
